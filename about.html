Acerca de EduLink

EduLink es una plataforma web interactiva diseñada para la distribución y visualización de recursos educativos en tiempo real. Este proyecto demuestra la aplicación práctica de redes, protocolos de comunicación y arquitectura cliente-servidor.

🛠️ Tecnologías Utilizadas

  • HTML5: Estructura semántica de la aplicación y maquetación interactiva.
  • CSS3: Diseño premium con gradientes modernos, efectos de desenfoque de fondo (glassmorphism) y adaptabilidad responsiva (Mobile First).
  • JavaScript (Vanilla): Lógica cliente, control de la sesión del usuario (localStorage), enrutamiento dinámico de parámetros y manipulación del DOM.
  • Protocolo HTTP/HTTPS: Transferencia de datos segura entre el cliente web y el servidor de archivos.

📡 Conceptos de Teleprocesos Aplicados

  • Modelo Cliente-Servidor: El navegador web (cliente) realiza peticiones HTTP para obtener recursos HTML, CSS, JavaScript y guías en formato PDF alojadas en el servidor.
  • Transmisión de Datos: Envío de flujos de bytes (streaming) de documentos y recursos dinámicos.
  • Autenticación de Sesión: Validación del lado del cliente y almacenamiento seguro del estado de sesión mediante tokens o almacenamiento web local.
  • Streaming Multimedia y Visualización Directa: Integración de lectores de PDF y visores embebidos para evitar la interrupción de la experiencia del usuario.
